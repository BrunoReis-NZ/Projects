Shopping List Project - HTML, CSS, JavaScript

Project description:

This is a simple shopping list project that I created using HTML, CSS, and JavaScript. The project is a single page application that allows the user to add items to a list, filter the list, and clear the list. The project is fully responsive and works on all screen sizes. The project is also accessible and works with screen readers. The project demonstrates my ability to use HTML, CSS, and JavaScript to create a fully functional web application.
